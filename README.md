======
Spinel: A fast web MVC framework based on BRB

Still documenting how to use it, please be patient ...

Features includes:
- Backbone
- requirejs
- handlebars
- PubSub
- Less
- Base Class and Mixins for model, view, router
- Session management
- Login (both page or modal), register
- notification
